 Basic Structure of an HTML Document An HTML document typically consists of the following structure: DOCTYPE Declaration: Specifies the HTML version being used. html Element: The root element that contains all other elements. head Element: Contains meta-information about the document, such as the title and links to stylesheets. body Element: Contains the content of the document, including text, images, links, and other media.
