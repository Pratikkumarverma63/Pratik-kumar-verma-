History of HTML

HTML was created by Berners-Lee in 1991 to facilitate the sharing and structuring of documents on the World Wide Web. Over the years, HTML has undergone several revisions, with HTML5 being the latest version, introducing new elements and APIs for modern web development.

Basic Structure of an HTML Document

An HTML document typically consists of the following structure:

  • DOCTYPE Declaration: Specifies the HTML version being used.
  • html Element: The root element that contains all other elements.
  • head Element: Contains meta-information about the document, such as the title and links to stylesheets.
  • body Element: Contains the content of the document, including text, images, links, and other media.

Common HTML Elements

Some commonly used HTML elements include:

  • Headings: <h1> <h6> for different levels of headings.
  • Paragraphs: <p> for text paragraphs.
  • Links: <a> for hyperlinks.
  • Images: <img> for embedding images.
  • Lists: <ul> for unordered lists and <ol> for ordered lists.
  • Tables: <table>, <tr>, <td>, and <th> for tabular data.

Conclusion

HTML is a fundamental technology for building web pages and applications. Understanding its structure and common elements is essential for anyone interested in web development.
